# DS340W_G26_ResearchProject
This project extends the parent paper by Alhamadi (2025) to investigate how part-time employment, combined with lifestyle factors such as sleep hours, diet, screen time, and exercise, affects academic performance among university students. 

In our research, we plan to use a lifestyle-focused student dataset, apply machine learning models such as XGBoost and LightGBM with SMOTE to address class imbalance, and use SHAP for explainability. Thus, we also engineer 5 novel features that are not present in the parent paper. 

# Our Research Question:
Can machine learning trained on lifestyle and employment data predict students' academic performance and reveal how part-time work shapes student well-being?
