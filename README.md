## 📋 Project Overview

This project investigates how part-time employment and lifestyle habits affect academic performance among university students. We extend the parent paper by adding five engineered features, SMOTE for class imbalance, SHAP explainability, and two novelty contributions — RFE feature selection and Optuna hyperparameter tuning.

## 🚀 How to Run the Code

**Step 1 — Open Google Colab**

Go to [colab.research.google.com](https://colab.research.google.com)

**Step 2 — Open the notebook from GitHub**
- Click **File** → **Open notebook**
- Click the **GitHub** tab
- Paste this URL: `https://github.com/AmeeraSab/DS340W_Group26_ResearchProject`
- Select `our_project_code/DS340W_Group26_Novelty.ipynb`

**Step 3 — Run all cells in order**
- Click **Runtime** → **Run all**
- Or manually run each cell from Cell 1 to Cell 21 in order
- The dataset loads automatically from GitHub, so no file upload needed

**Step 4 — Expected outputs**
- All charts will be generated and auto-downloaded
- Final results will be printed in Cell 17
- Novelty results will be printed in Cells 19 and 20
